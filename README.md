# HubSpot Leads Checker

## Required Files

### 1. HubSpot Deals (XLSX export)
- Export "alle deals" list as XLSX from HubSpot
- Must contain:
  - Associated Contact
  - Associated Company

### 2. Deal Alignment (XLSX export)
- Export "Deal Alignment" list as XLSX from HubSpot
- Must contain:
  - Domain-Name des Unternehmens
  - Unternehmensname

### 3. New Leads (XLSX)
Required columns:
- Vorname
- Nachname
- Email
- Unternehmen
